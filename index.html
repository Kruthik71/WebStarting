JavaScript

1) Developed by netscape then acquired by ECMA to make it standardized.

2) ECMA has GDPR(General Data Protection Regulation) a committe which oversees the protection of data of users.

3) Only variant type is available in javascript which is used as (var).

4) In JS the keywords are case-sensitive.

5) Method Overloading can be done in JS greeting(),greeting(101),greeting(101,"Kruthik") like this and can have respective functions.

5.1) function greeting() -> here function is a reserved keyword

5.2) var greeting=function(){ . . . } -> is called anonymous function

5.3) ( function() { . . . . }) (); -> is a IIFE stands for Immediaely invoked function expression which can be used only once in a running script, Example. (Cookies,Notification sending,Greeting to a wesite)

19) Browser Object Model and Document Object Model

19.1) Here if we see the above image is a Heirarchial structure represnted as a Tree.

19.2) Trees in Context of data structures has levels to it.

19.3) The Window Object is at Level-0 whereas links,image,forms are at Level-2

19.4) So, the BOM is called Level-0 DOM and Document & their children are referred as DOM at Level-2

19.5) Window object is the parent to all objects.

19.a) Window Object provided by the browser itself.

19.a.1) Represents the browser window itself.

19.a.2) It has load and unload events, open() and close() methods

19.a.3) It has properties such as innerHeight,innerWidth,status to manipulate and change the window size an state.

19.a.4) Window Object Demo -> Window Object

19.b) Location Object

19.b.1) Contains information about the current URL.

19.b.2) It has a reload() method

19.b.3) It has properties such as href,hash,port,protocol,hostname that can be accessed.

19.b.4) Location Object Demo -> Location Object

19.c) Navigator Object

19.c.1) Contains information about the browser.

19.c.2) It has a javaEnabled() method

19.c.3) It has properties such as appName,appVersion,userAgent that can be accessed.

19.c.4) Navigator Object Demo -> Navigator Object

19.d) History Object

19.d.1) Contains URL's visited by the user (Read only properties and Methods).

19.d.2) It has back(),forward(),go() methods

19.d.3) It has a length property that can be accessed.

19.e) Document Object

19.e.1) Represents the HTML document loaded into the browser.

19.e.2) It has open(),close(),write() methods

19.e.3) It has properties such as lastModified,doctype that can be accessed.

19.e.4) Document Object Demo -> Document Object

19.f) Image Object -> represeting an embedded image and has properties (src,alt).

19.g) Link Object -> represents HTML hyperlink has properties (href,target), where target can have following values (_blank,_self(default),_parent)
