1) Few things to remeber

1.1) Month is specified in (0-11) numerics when setFullYear() function is used which accepts 3 parameters.

2) We can add functions to Date Object itself and make Use of them using Prototype Keyword.


        // Prototyping by extending it for Date Object
        Date.prototype.getDayNameNotaInbuiltFucntion=function(){
            var days=["Sun","Mon","Tue","Wed","Thu","Fri","Sat"];
            return days[this.getDay()];
        }
        var dt=new Date();
        dt.setFullYear(2001,2,4);
        alert(dt.getDayNameNotaInbuiltFucntion());
